My app errors when logging in and says "Canceled"

Users will see this error if using an iOS (Apple) device that have a beta installed. Either iOS 11.4.1 beta or iOS 12 betas will cause the problem. Note that although the 11.4.1 beta causes the issue, the actual 11.4.1 release does not.

To resolve the issue, you will need to uninstall any beta iOS software and return to using the normal release.
