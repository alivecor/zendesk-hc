Cardiac Rehab Centers Promo Code TD6PWMCZ

 

We printed thousands of brochures for patient bags at Cardiac Rehab Centers. These brochures contained a promotional code which entered will give patients a free CarryPod with their purchase of Kardia Mobile. The promotional code for all of these brochures is the same and is TD6PWMCZ. 

In order for the code to work, a customer must add both a Kardia Mobile AND a CarryPod to their cart. Once they select checkout, they can enter the promotional code of TD6PWMCZ, and it will subtract the cost of the carrypod.
