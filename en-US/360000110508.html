How Do I get a User Manual?

To download or print a user manual, got to www.alivecor.com. Scroll to the bottom of the screen, and select "User Manual" under the Support section. Select whether you want an Android User Manual or an Iphone User Manual, then select the desired language. This will open the user manual as a PDF in a new tab in your web browser. You can now save or print the user manual.Â
