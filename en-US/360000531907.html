I need to cancel my order

If a user asks to cancel their order, follow the bellow process:

 
1. Check shopify to see if it has been fulfilled
  • If it has already been fulfilled give the client the RMA info, and let them know that they will need to return the device back to us for a refund. 
2. If it hasn't yet been fulfilled, email the below DCL contacts with the Shopify order number, and ask if they can cancel the account. 
  • ave-20400-ff@dclcorp.com, ave-20400-am@dclcorp.com, borisb@dclcorp.com. 
3. If they respond and say that they have canceled it, go ahead and cancel the order in Shopify, which will automatically refund the user. 
4. If they respond and say that it has already been fulfilled the user will have to through the RMA refund process.
