Will the Kardia app Instant Analysis still detect Possible Atrial Fibrillation if the user has another existing heart arrhythmia?

If a user is experiencing Possible AFib and other abnormalities at the same time, the Instant Analysis will still list "Possible AFib".  The algorithms are written to show Afib if it is detected, even if other arrhythmia are present in the recording. (Per confirmation from Daniel)
